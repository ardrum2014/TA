    ☕️ 歡迎使用「南寧咖啡館」智慧教師小手帳

    本系統專為教師課堂管理與互動教學設計。全站採用溫暖手寫質感的芫荽字體 (Iansui)與咖啡廳舒壓配色，支援完全離線使用、本機永久資料儲存與全站雲端備份還原！

    0. 今日班務與任務指揮中心

    (點擊展開)
    🌟 功能特點：
    • 📅⏰ 獨立同排大字體展示：頂部第一排高亮卡片同時顯示 `📅 今日日期 (如 2026/07/23 週四)` 與 `⏰ 實時秒數 (如 16:00:07)`，字體加大極致清晰。
    • 📚 課節與狀態動態導航：實時讀取個人週課表，自動顯示當前進行課節及下一節預告。
    • 👥 班級欠交概況與選單摘要 Modal：提供班級切換下拉選單（包含「全校所有班級」與單一班級），點選「🔍 瀏覽摘要」可彈出跨學科未繳名冊並一鍵複製。
    • 🔔 任務時間提醒 (含日期與 ⚠️ 已逾期檢核)：支援選擇預訂日期與時間，系統自動比對時間，若任務超過預訂時間未完成自動顯示「⚠️ 已逾期」警示紅框。
    • ⚡ 極速快捷工具發射台：包含催繳單、分析儀表板、幸運輪盤、計時器與加分榜快捷發射按鈕。
    🛠️ 操作流程說明：
    1. 掌握目前時間與課程：登入網站預設進入今日儀表板，頂部第一排卡片即可一覽今日日期與實時秒數，右側可查看當前課節與下一節預告。
    2. 切換檢視全校/單班缺交摘要：在「班級動態概況」右上方選單切換至「全校所有班級」或指定班級，點選「🔍 瀏覽摘要」即可跳出彈窗瀏覽欠交名冊並點選「複製此摘要報告」。
    3. 建立任務與時效追蹤：點擊「+ 新增提醒任務」，輸入任務名稱並挑選 `預訂日期` 與 `提醒時間`。若任務超過預訂時間未完成，卡片會自動標示「⚠️ 已逾期」；完成後勾選核取方塊即可劃線歸檔。
    4. 快捷工具發射：點選極速發射台上的 5 個工具按鈕，可秒級切換至催繳單、輪盤、計時器、加分榜等功能。

    1. 課堂提醒與計時器

    (點擊展開)
    🌟 功能特點：
    • 四大切換模式：課堂流程模式、倒數計時、正計時碼錶、實時大時鐘。
    • 快捷時間點擊：提供 1分、3分、5分、10分、15分一鍵啟動快捷按鍵。
    • 分段計圈與一鍵歸零：在碼錶模式下支援記錄多次分段時間，並提供一鍵重設歸零。
    • 全域鍵盤快捷鍵：支援空白鍵與常用字母鍵控，免去繁瑣的滑鼠點擊移動。
    • 投影放大模式：全螢幕高亮展現，按 ESC 鍵可快速切換回主畫面。
    🛠️ 操作流程說明：
    1. 選擇模式與設定：在頂部切換為「課堂提醒」、「倒數計時」等模式，點選快捷按鈕或調整拉桿，點擊「開始計時」。
    2. 正計時碼表與快捷鍵：切換至「正計時碼表」，點選「開始」。計時期間可點選「記錄時間」或按鍵盤 **Enter / L** 鍵記錄當下分段時間；按下 **Space (空白鍵)** 可快速開始或暫停計時；按下 **R** 鍵或重置按鈕可一鍵歸零時間與紀錄清單。
    3. 啟用全螢幕投影：點擊放大圖示 進入全螢幕簡報視圖；若要關閉，按下鍵盤左上角的 ESC 即可退出。
    4. 匯入課堂流程：下載 Excel 範本填寫流程（含時間、時長、主題、內容），點擊「匯入課堂流程」上傳即可依序自動提醒授課進度。

    2. 抹茶幸運輪盤抽籤與隨機任務卡

    (點擊展開)
    🌟 功能特點：
    • 學生與小組雙重抽籤：除了抽單一學生，更支援「抽分組」模式。
    • 自適應分組連動：抽分組時會自動讀取當前班級的實際小組組數與自訂組名，智慧分割輪盤扇區。
    • 🃏 隨機任務/問答卡連動：提供 `[x] 抽中時連動翻開隨機任務卡` 複選框（預設關閉保留純抽籤），抽中時自動翻開挑戰卡。
    • ✏️ 客製化任務卡編輯：點擊「✏️ 編輯任務卡」可跳出彈窗自訂隨機問答或加分題目庫。
    🛠️ 操作流程說明：
    1. 選擇抽籤類別與帶入名單：選擇「抽學生」或「抽小組」模式，點擊「帶入班級名冊」載入名單。
    2. 設定隨機任務卡：如需課堂互動，可勾選 `[x] 抽中時連動翻開隨機任務卡`；若要修改任務題目，點選旁邊的「✏️ 編輯任務卡」於彈窗中修改題庫並儲存。
    3. 旋轉開獎：點擊輪盤任意處或「開始旋轉抽籤！」，輪盤轉動開獎，開獎視窗會同時顯示中籤者與隨機任務卡！

    3. 智慧座位表、A4海報列印與學生戰報

    (點擊展開)
    🌟 功能特點：
    • 🔒 唯讀與 ✏️ 編輯雙模式：預設為唯讀防誤觸模式，點擊「✏️ 編輯模式」解鎖名牌對換。
    • 🖨️ 列印 A4 海報：一鍵將班級座位表自動排版為超大字體、美觀邊框與講台標記的 A4 橫向高清海報，適合張貼於公佈欄。
    • 🪪 學生個人綜合戰報：學生名牌角角附帶 `🪪` 微型按鈕，點擊彈出戰報（含點數、座位座標、組長標記 👑、欠交作業明細與一鍵複製）。
    • ⚙️ 條件排座設定：設定「禁止相鄰(8向)」、「必須相鄰(小老師)」、「指定前後排」，由 CSP 限制解算器自動排座。
    🛠️ 操作流程說明：
    1. 排座與編排：點選「✏️ 編輯模式」解鎖。點選「⚙️ 條件設定」新增限制，點擊「隨機編排座位」自動生成，或拖曳名牌微調，點選「💾 儲存座位表」。
    2. A4 教室張貼海報列印：點選工具列的「🖨️ 列印 A4 海報」，系統會開啟 A4 橫向高清列印視圖，點擊列印即可直接張貼於教室。
    3. 查閱學生綜合戰報：點選名牌右上角的 `🪪` 微型圖示，彈出個人綜合戰報，可點選「複製學生綜合學習報告」發送給家長。

    4. 隨機與彈性分組工具與 A4 海報列印

    (點擊展開)
    🌟 功能特點：
    • 隨機分組與拖曳調整：支援依組數/人數一鍵隨機分組，分組與組員順序自動儲存。
    • 組別管理與組長設定：支援「手動新增小組」、點擊修改組名，並可指派唯一組長 👑。
    • 🖨️ 列印 A4 海報：一鍵將小組名冊與組長表自動排版為 A4 橫向海報輸出張貼。
    • 同步至分組加分榜：一鍵將分組結構對齊同步至競賽加分榜。
    🛠️ 操作流程說明：
    1. 執行自動分組：選擇「依總組數」或「每組人數」，點選「開始自動分組」。
    2. 拖曳與組長指派：拖曳名牌微調組員，點擊 👑 圖示指派組長。
    3. A4 列印與同步加分：點選「🖨️ 列印 A4 海報」進行張貼輸出；點選「同步至分組加分榜」對齊得分資料。

    5. 課堂加分與小組競賽榮譽榜

    (點擊展開)
    🌟 功能特點：
    • 🏆 依得分名次排行榜排序：支援在 `[ 🔢 依座號/組別預設順序 ]` 與 `[ 🏆 依得分名次排行榜 ]` 之間一鍵切換。
    • 🥇🥈🥉 榮譽名次勳章：自動計算高低分名次，為前三名頒發 **🥇 第1名**、**🥈 第2名**、**🥉 第3名** 榮譽勳章與金邊高亮外框。
    • 個人/小組跨班隔離：卡片式加扣分，伴隨音效與紙花慶祝。
    • 整合匯出 Excel 成績單：一鍵匯出個人得分與小組得分合併總表。
    🛠️ 操作流程說明：
    1. 切換加分與排行榜視圖：切換「個人加分」或「分組競賽」標籤。點選頂部的「🏆 依得分名次排行榜」按鈕，卡片將自動按最高分起算排序，並標示 🥇🥈🥉 榮譽勳章！
    2. 點數加扣分：點擊卡片上的「+1」或「-1」按鈕進行分數調整，加分伴隨音效與紙花。
    3. 匯出與歸零：點選「匯出加分 Excel」下載總表；點選「點數歸零」可選擇歸零當前班級或全數班級。

    6. 作業與習作追蹤

    (點擊展開)
    🌟 功能特點：
    • 四態標籤與精準統計：未繳 ❌ ➔ 已繳 🟢 ➔ 待訂正 🟡 ➔ 補繳 🔵，已繳人數統計精確排除補繳。
    • 一鍵複製未繳名單：產出整理好的未繳學生名冊，方便貼至聯絡簿或班級群組。
    • 檢核彈窗資訊擴展：檢核彈窗中的學生資訊整合顯示學號、幹部及備註。
    🛠️ 操作流程說明：
    1. 建立與選取作業項目：點擊「建立新作業」輸入名稱並儲存，或在下拉選單中選取現有作業。
    2. 登記繳交狀態：點擊學生欄位中的狀態標籤，可在「未繳 ❌」、「已繳 🟢」、「待訂正 🟡」、「補繳 🔵」四種狀態中按順序循環切換。
    3. 一鍵催繳複製：作業收齊期間，點選「一鍵複製未繳名單」，系統會將已整理好的未繳名條複製至剪貼簿（已繳人數會自動精準排除補繳者，不造成統計誤差）。

    7. 課程與作業追蹤整合管理

    (點擊展開)
    🌟 功能特點：
    • 卡片與手風琴展開：科目卡片預設為收合狀態，點擊卡片標題任何空白處即可展開。
    • 📋 跨學科一鍵催繳單：跨越所有科目，自動以座號排序彙整所有學生的 `未繳 ❌` 與 `待訂正 🟡` 明細，生成方便貼至聯絡簿的催繳文字，支援就地修改與一鍵複製。
    • 📊 座號繳交熱點圖 (Heatmap)：點擊作業標籤旁的 `📊` 圖示可展開座號小方格。小方格以紅（未繳）、綠（已繳）、黃（待改）、藍（補繳）標記狀態。**點選座號格即可就地極速切換狀態**，省去彈窗操作。
    • ✏️ 就地重新命名（無損保留）：點擊章節標題或作業標籤旁邊的 `✏️` 編輯按鈕，即可隨時微調名稱，並 **100% 完整保留現存學生的歷史登錄紀錄**。
    • 📋 快速套用結構範本：提供「語文科經典 (14課)」、「數學科 (10單元)」、「科技與藝能科 (6單元)」範本，一鍵快速建置全學期作業與章節架構。
    • ⚠️ 繳交預警分析儀表板：自動列出全班「欠交 3 件以上作業的學生名單 (欠交大戶排行)」與「各科整體授課進度條」，便於導師個別關懷。
    🛠️ 操作流程說明：
    1. 建立科目與套用範本：點選「➕ 新增科目」輸入科目名稱（直接點擊標題文字為靜態以防誤觸，如需修改請點擊右側的「✏️ 編輯」鈕）。展開科目卡片後，可點選右上角的「📋 套用範本」一鍵快速建立整學期的課堂作業結構。
    2. 進行座號熱點快速登記：點擊作業名稱（如課堂練習）可以打開完整名冊大彈窗；如果需要快速修改個別座號狀態，可點選標籤旁的 `📊` 網格圖示展開座號熱點圖，直接按點座號方格進行循環狀態切換。
    3. 修改章節與作業名稱：若需微調文字，可點擊章節右側的「✏️」圖示或作業 Pill 中的「✏️」按鈕進行無損更名（右側的刪除按鈕已拉開安全間距並附帶視覺分隔線，以防止誤觸）。
    4. 產出全科催繳單與預警分析：點選控制列最頂端的「📋 產生全科催繳單」可自動生成便於發送的未繳總清單；點選「⚠️ 繳交狀況分析儀表板」則可隨時檢視各科進度與缺交大戶排行榜。

    8. 名條管理與全站雲端備份/還原

    (點擊展開)
    🌟 功能特點：
    • 手動編輯與排序座號：在網頁上可直接修改學生姓名、學號、幹部、備註四欄，支援拖放或按鈕重新排序，存檔後自動重編座號並全站連動。
    • Excel 名條多班級匯入：支援解析 CSV, XLS, XLSX，相容南寧高中公假系統格式。
    • 全站備份與還原 (`.json`)：一鍵備份全站資料至本機，更換電腦時一鍵匯入即可全數還原！
    🛠️ 操作流程說明：
    1. Excel 多班名冊匯入：下載名條範本填寫，或直接解析南寧高中公假系統 CSV/XLS 表單，點選「選擇名條檔案」即可秒級載入全班名冊。
    2. 手動增修與排序名冊：點擊「✏️ 編輯學生資料與排序」，可直接就地修改學生的學號、幹部與備註；透過滑鼠拖放或點選上下按鈕可調整學生順序，點選「💾 儲存修改」後，系統會自動按最新排序重編 1..N 的座號並全站生效！
    3. 全站備份與還原移轉：更換電腦或定期存檔時，點選側邊欄底部的「備份全站」下載 `.json` 還原檔；在新電腦中點選「📥 匯入還原」選擇此檔案，即可一秒將分組、座位、加分、進度等所有班級資料全數還原。

    常見問題解答 (FAQ)

    Q1: 重新整理或關閉瀏覽器後，我設定的資料會遺失嗎？
    答：不會！所有學生名單、座位表、分組、加分與活動流程均安全存放在您瀏覽器的本機儲存區 (LocalStorage)，重新開啟瀏覽器皆會自動載入。
    Q2: 如果我要更換電腦，該如何移轉這裡的所有資料？
    答：請在舊電腦點擊側邊欄底部的「備份全站」，將產生的 `.json` 備份檔傳至新電腦，在新電腦點擊「📥 匯入還原」並選取該備份檔，即可全數還原！
    Q3: 如何快捷退出全螢幕投影？
    答：在任何全螢幕投影畫面中（如課堂提醒、計時器、輪盤），按下鍵盤上的 ESC 鍵即可隨時切換回主畫面。
